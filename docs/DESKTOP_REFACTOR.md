# Raphael 데스크톱 앱 리팩토링 계획

> 작성일: 2026-04-17 | 기준 버전: v0.1.12 | 총 기능: 117개

---

## 1. 제거 대상 (6개)

| # | 대상 | 현황 | 이유 |
|---|---|---|---|
| R1 | **훅(Hooks) 탭** | API 있으나 UI 미완성 | CLI `raphael watch`만으로 충분, 데스크톱에서 파일 워처 설정은 비직관적 |
| R2 | **업데이트(Update) 탭** | git pull + pip install | Tauri 자동 업데이트가 앱 바이너리 처리, 이 탭은 개발자 전용 (CLI `raphael update`) |
| R3 | **파일 변환 도구(Tools) 탭** | 대시보드 내 변환 UI | 채팅에서 에이전트가 도구로 수행 가능, 별도 GUI 불필요 |
| R4 | **라우팅 추천 모델 적용 버튼** | 하드코딩된 추천값 | 사용자 환경과 무관한 추천은 혼란 유발 |
| R5 | **A/B 테스트 잔존 API** | UI 삭제됨, daemon 엔드포인트+api.ts 타입 잔존 | 사용처 없음 |
| R6 | **AbResult 타입** (api.ts) | UI 미사용 | 코드 정리 |

## 2. 통합 대상 (설정 탭 10→6개)

| # | 변경 | Before | After |
|---|---|---|---|
| M1 | **모델 + 라우팅 통합** | 모델 탭, 라우팅 탭 (각각) | → **모델** 탭 (하위에 라우팅 섹션) |
| M2 | **서버 + 풀 + 보안 통합** | 서버 탭, 풀 탭, 보안 탭 (각각) | → **서버** 탭 (Ollama 설정 + 다중 서버 풀 + 허용 경로/시크릿) |
| M3 | **프로필 → 에이전트 탭 하위** | 프로필 탭 (fact 관리) | → 에이전트 탭 하단 "사용자 프로필" 섹션 |

```
에이전트 (+ 프로필) | 스킬 | 모델 (+ 라우팅) | 서버 (+ 풀 + 보안) | RAG
```
